# Fork Plans
The plans for the fork of [react-markdown](https://remarkjs.github.io/react-markdown/)

- Images
ex:
blah blah blach
{image src='/image.png'}
images must be in public or public/markdown-images
- Add warning, error, etc vlocks for info
ex:
= ERROR
Error text
=
= WARNING
Error text
=
= INFO
Error text
=
- Have mutiple code blocks with different languages
ex: 
```javascript block-1
example code
```
```bash block-2
example code
```
```C# block-3
example code
```
This would create 3 different blocks the user can select


This fork is planned to only be used on CodexFlow but it will be it's own repo once I start working on it.
My fork will be MIT like the react-markdown project is already.
